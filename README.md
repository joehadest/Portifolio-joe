# < Joe.dev /> - Portfólio Pessoal

![Banner do Projeto](./assets/banner.png)

## 🚀 Sobre o Projeto

Portfólio profissional moderno e responsivo, desenvolvido com foco em design minimalista e experiência do usuário. Apresenta uma interface interativa com animações suaves e efeito glass morphism.

### ✨ Destaques

- Design moderno com efeito glass morphism
- Animações suaves e interativas
- Totalmente responsivo
- Performance otimizada
- Interface intuitiva

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome
- Google Fonts (Poppins)

## 🔥 Funcionalidades

- **Header Dinâmico:** Transparente e com efeito blur ao rolar
- **Efeito de Digitação:** Animação de texto na seção hero
- **Cards Interativos:** Projetos com hover effects e animações
- **Barras de Progresso:** Skills com animações de carregamento
- **Menu Mobile:** Interface adaptativa com animações suaves
- **Scroll Suave:** Navegação fluida entre seções
- **Micro-interações:** Feedback visual em botões e links

## 📱 Responsividade

- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)

## ⚡ Performance

- Imagens otimizadas
- CSS minificado
- Carregamento assíncrono
- Lazy loading de imagens

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/joehadest/portfiolio-joe.git
```

2. Navegue até o diretório:
```bash
cd portfiolio-joe
```

3. Abra o `index.html` no navegador
